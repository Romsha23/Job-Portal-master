# Job-Portal-master
# 💼 Full Stack Job Portal

A modern and responsive **Full Stack Job Portal** built with **React.js**, **Tailwind CSS**, **Supabase**, **Clerk Authentication**, and **Shadcn UI**. The platform connects job seekers with recruiters through a clean, secure, and user-friendly interface.

## 🚀 Live Demo

🔗 https://job-portal-azure-five.vercel.app/

## ✨ Features

- 🔐 Secure authentication with Clerk
- 👤 Role-based access for Job Seekers and Recruiters
- 📝 Recruiters can post and manage job listings
- 🔍 Search and browse available jobs
- 📄 View detailed job descriptions
- 📌 Apply for jobs seamlessly
- 🏢 Company profile management
- 📱 Fully responsive UI
- ⚡ Fast and optimized performance
- 🎨 Modern interface using Shadcn UI

## ⚙️ Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
```

## 📦 Installation

```bash
git clone <repository-url>

cd job-portal

npm install

npm run dev
```
